DESCRIPCIÓN:
Este proyecto implementa un juego de Tres en Raya en Python, donde un humano juega como X contra una IA entrenada con Q-Learning que juega como O.

INSTALACIÓN Y USO:
Instala PyGame:
pip install pygame

Entrena la IA (ejecuta primero):
python entrenamiento.py

Juega contra la IA:
python interfaz.py
